# 资助 V2Ray 发展

Project V 是一个非营利项目，它的所有代码均公开，软件可以自由使用，不受限制。如果您喜爱本项目，可以通过下面的方式请作者喝一杯咖啡。

* 所有捐款的用途将不被公开；
* 您的捐款是对 Project V 的无偿贡献，和 Project V 提供的技术和服务没有直接关联；

{% hint style='tip' %}
感谢您对 Project V 的热爱和支持。我们会对每一份捐赠发送一份确认函，请务必在捐赠时留下您的邮件地址，或留言注明不需要确认。
{% endhint %}
